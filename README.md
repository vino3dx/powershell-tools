## 📄 CSV提取每行保存为TXT（首列为名字）.ps1

### 功能
- 将一个 CSV 文件中的每一行转换为一个独立的 TXT 文件。
- **第一列**作为文件名（自动替换非法字符）。
- **第二列**作为 TXT 文件的内容。
- 输出文件夹会自动创建，位于 CSV 文件所在目录下，文件夹名与 CSV 文件名一致。

### 使用方法
```powershell
.\CSV提取每行保存为TXT.ps1 -inputFile "C:\路径\文件.csv"
