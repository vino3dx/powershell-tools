# PowerShell 工具脚本集

本仓库收录了一些实用的 PowerShell 脚本，旨在简化文件处理与自动化管理流程。

---

## 📄 1. CSV 提取每行保存为 TXT

### 功能描述

* **自动化拆分**：将 CSV 文件的每一行记录提取出来，并分别保存为独立的 `.txt` 文件。
* **智能命名**：自动读取 **第一列** 作为文件名，并自动过滤文件名中的非法字符。
* **内容映射**：自动读取 **第二列** 作为 TXT 文件的正文内容。
* **自动建卷**：在 CSV 同级目录下自动创建与原文件名一致的文件夹。

### 使用方法

```powershell
.\CSV提取每行保存为TXT（首列为名字）.ps1 -inputFile "C:\路径\您的文件.csv"

```

## 📂 2. 设置文件夹时间戳

### 功能描述

* **时间校准**：一键修改目标文件夹的 **创建时间**、**修改时间** 和 **访问时间**。
* **批量管理**：适用于需要统一归档时间或整理旧数据的场景。

### 参数说明

| 参数 | 描述 | 示例 |
| --- | --- | --- |
| `-folder` | 目标文件夹的完整路径 | `"D:\Backups\2026"` |
| `-date` | 目标日期（格式：YYYY-MM-DD） | `"2026-03-04"` |

### 使用方法

```powershell
.\设置文件夹时间戳.ps1 -folder "C:\您的路径" -date "2026-03-04"

```

## ⚙️ 系统要求与环境

* **环境**：Windows PowerShell 5.1 或更高版本。
* **编码**：建议使用 UTF-8 编码以支持中文路径。
* **权限**：如果运行受限，请先执行：`Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`。
